# Active-Window-Speech-To-Text-Console

Gerekli API projeye dahil edilmelidir.

1) Project menüsünden "Manage NuGetPackages" seçeneğine ulaşın
2) Browse sekmesindeyken arama kısmına API adı yazın ve install komutu verin
  a) API 1: Google Cloud Speech V1
  b) API 2: AutoIt
3) Gerekli kurulum işlemi için install komutu verin ve kurulumun tamamlanmasını bekleyin
4) Projeye dahil ettiğimiz API kullanmak için using satırlarını da eklerseniz problem çözülecektir
  a) using AutoIt;
  b) using Google.Cloud.Speech.V1;
5) Artık proje çalıştırmaya hazır.
